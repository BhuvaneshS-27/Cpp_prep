For future updates

Whenever you add or modify files:

git add .
git commit -m "Describe what changed"
git push

to delete a file from repository:
git rm --cached "Code io/L41_maps.exe"
git commit -m "Remove executable files"
git push

to pull changes from repository to local:

git pull origin main --rebase
